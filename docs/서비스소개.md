# M2 광의통화 대시보드 서비스 소개

## 1. 서비스 개요

**M2 광의통화 대시보드**는 한국은행 경제통계시스템(ECOS)에서 제공하는 M2(광의통화) 공공데이터를 수집·시각화하는 웹 서비스입니다.

접근 허용 리스트 기반의 인증 체계를 통해 승인된 사용자만 데이터를 열람할 수 있습니다.

---

## 2. M2(광의통화)란?

M2(광의통화, Broad Money)는 한 나라의 통화량을 측정하는 지표 중 하나입니다.

| 구분 | 포함 항목 |
|------|-----------|
| M1(협의통화) | 현금 + 요구불예금 + 수시입출금식 저축성예금 |
| M2(광의통화) | M1 + 2년 미만 정기예적금 + 머니마켓펀드(MMF) 등 |

M2는 중앙은행(한국은행)의 통화정책 및 경기 분석의 핵심 지표로 활용됩니다.

---

## 3. 주요 기능

### 3.1 Google OAuth 인증
- Google 계정을 통한 소셜 로그인
- NextAuth.js 기반 JWT 세션 관리

### 3.2 접근 허용 리스트(Allowlist) 제어
- Turso DB에 등록된 이메일 계정만 대시보드 접근 허용
- 미등록 사용자는 접근 차단 페이지로 리디렉션

### 3.3 M2 데이터 시각화
- 최근 3년간 월별 M2 광의통화 잔액 추이 라인 차트
- 최신 M2 잔액, 전월 대비·전년 동월 대비 증감률 요약 카드
- 월별 상세 데이터 테이블 (역순 정렬)

### 3.4 데이터 캐싱
- ECOS API 호출 결과를 Turso DB에 캐싱
- API 장애 시 캐시 데이터 자동 사용

## 4. 시스템 아키텍처

```
사용자 브라우저
    │
    ▼
Vercel (Next.js App Router)
    ├── 인증: NextAuth.js + Google OAuth
    ├── 미들웨어: 접근 허용 리스트 검증
    ├── API Route: /api/m2
    │       ├── ECOS API 호출 (한국은행)
    │       └── 캐시 R/W
    └── Turso DB (libSQL)
            ├── allowed_users (허용 사용자 목록)
            └── m2_cache (M2 데이터 캐시)
```
